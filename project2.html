3D Dodecahedron for Burning Man 2024

Engineered a 3D dodecahedron structure for Burning Man 2024 featuring exterior LED animations controlled by an Arduino Teensy. The project integrates custom LED animation simulation software with Blender 3D and a low-code video projection algorithm to create a seamless, dynamic visual experience.

Technical Details

The LED animations were developed using custom simulation software that allowed for rapid prototyping and real-time adjustments. Blender 3D was employed to design the structure and ensure that the LED projections align perfectly with the facets of the dodecahedron. The low-code video projection algorithm simplifies the animation workflow, while the Arduino Teensy provides robust, real-time control of the LEDs.

This innovative pipeline has streamlined the design process and pushed the boundaries of interactive installations, making it a standout project in my portfolio.
